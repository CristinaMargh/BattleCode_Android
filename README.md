# BattleCode

BattleCode is an educational quiz-based application designed to help users improve their knowledge in programming, algorithms, and software engineering concepts.
The application provides timed question sessions, difficulty-based scoring, and persistent user progression tracking.

## Features

- User authentication (login/logout)
- Quiz sessions with a fixed number of questions and time limits
- Difficulty-based scoring system
- Dynamic question selection by topic or difficulty
- Persistent storage of users, questions, and scores
- Scoreboard / ranking system between users
- Fragmentation : reusable fragment foar leaderboard
## Technologies

- **Platform:** Android
- **Language:** Kotlin
- **Architecture:** Separation of UI, logic, and persistence layers
- **Data Storage:** Local database
